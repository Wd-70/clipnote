# ClipNote 개발 로드맵

> **핵심 포지셔닝:** "편집 없이 클립, 바로 공유"
>
> 영상 편집을 몰라도 누구나 클립을 만들고 공유할 수 있는 서비스

---

## 타겟 사용자

| 세그먼트 | 설명 | 핵심 니즈 |
|---------|------|----------|
| **스트리밍 시청자/팬** (1차) | 치지직/트위치/유튜브 시청자 | 명장면 클립 + 간편 공유 |
| **스트리머/크리에이터** (1차) | 긴 방송에서 하이라이트 추출 | 빠른 클립 생성, 팬 공유 |
| **영상 편집자** (2차) | 편집 전 사전 구간 표시 | 타임스탬프 정리, 내보내기 |
| **교육/학습자** (2차) | 강의 핵심 구간 정리 | 구조화된 노트, 빠른 탐색 |

---

## 강점

1. **멀티 플랫폼 통합** — YouTube + 치지직 + Twitch를 하나의 앱에서 지원. 특히 치지직 지원은 한국 시장 차별화 포인트
2. **진입 장벽 제거** — 영상 편집 지식 불필요. URL 붙여넣기 → 시간 찍기 → 공유까지 3클릭
3. **AI 하이라이트 자동 분석** — Gemini AI로 영상 핵심 구간 자동 탐지, 수동 작업 시간 단축
4. **바이럴 내장 구조** — 공유된 클립을 본 사람이 자연스럽게 서비스로 유입

## 개발 Phase

---

### Phase 0: 기반 정비 (1~2주)

출시 가능한 최소 상태로 만든다.

- [ ] **결제 시스템** — Portone 실결제 연동 (포인트 구매)
- [ ] **소셜 로그인** — 네이버/카카오 로그인 추가
- [ ] **Rate Limiting** — AI 분석 등 비용 발생 API 보호
- [ ] **에러 모니터링** — Sentry 연동
- [ ] **핵심 테스트** — 타임스탬프 파싱, 클립 생성 로직 단위 테스트
- [ ] **검색 기능** — 프로젝트/클립 내 검색

---

### Phase 1: 공유 경험 완성 (3~4주)

**목표:** 공유 링크를 받은 사람이 "이건 뭘로 만든 거야?"라고 묻게 만든다.

#### 1-1. 공유 플레이어 고도화

- [ ] 공유 페이지에서 클립 구간만 이어서 재생하는 플레이어
- [ ] 클립 간 부드러운 전환 (페이드 또는 카운트다운)
- [ ] 현재 재생 중인 클립 하이라이트 표시
- [ ] 모바일 최적화 (세로 모드 쾌적 재생)

#### 1-2. 원클릭 클립 생성 플로우

- [ ] 로그인 없이 URL 붙여넣기 → 바로 에디터 진입 (게스트 모드)
- [ ] "시작" → 시청 → "끝" → 공유 링크 생성까지 3클릭
- [ ] 회원가입은 저장/관리 시점에 유도

#### 1-3. 공유 페이지 브랜딩

- [ ] "Powered by ClipNote" 워터마크 + 가입 유도 CTA
- [ ] OG 이미지 자동 생성 (영상 썸네일 + 클립 정보 합성)
- [ ] 카카오톡/트위터/디스코드 미리보기 최적화

---

### Phase 2: 자막 시스템 (4~6주)

**무료 사용자를 끌어들이고, 구독으로 전환시키는 핵심 기능**

#### 2-1. 기본 자막 (무료)

| 항목 | 내용 |
|------|------|
| STT 엔진 | Whisper API 또는 Google Speech-to-Text |
| 자막 형식 | 시간 동기화 텍스트 (SRT/VTT 호환) |
| 표시 방식 | 클립 재생 시 하단 자막 오버레이 |
| 편집 | 자막 텍스트 수정 가능 (오타 교정) |
| 지원 언어 | 한국어, 영어, 일본어 |
| 스타일 | 흰색 텍스트 + 반투명 검정 배경 (기본 1종) |

자막 생성 파이프라인:
```
클립 구간 확정
  → 해당 구간 오디오 추출 (서버사이드)
  → STT API 호출
  → 타임스탬프별 텍스트 반환
  → 사용자 편집 가능 상태로 저장
  → 재생 시 동기화 렌더링
```

#### 2-2. 프리미엄 자막 스타일 (구독)

> 구독 전환의 핵심 트리거.
> "자막 꾸미고 싶다" → 구독

| 카테고리 | 무료 | 구독 |
|----------|------|------|
| **폰트** | 시스템 기본 1종 | 10+ 프리미엄 폰트 |
| **색상** | 흰색 고정 | 자유 색상 + 그라데이션 |
| **배경** | 반투명 검정 | 투명/컬러/그림자/외곽선 |
| **위치** | 하단 고정 | 상단/중앙/자유 배치 |
| **애니메이션** | 없음 | 페이드인, 타이핑, 바운스 |
| **이모지/스티커** | 없음 | 자막에 이모지, 강조 효과 |
| **프리셋** | 없음 | "유튜브 밈", "뉴스 자막", "예능 자막" 등 |

#### 2-3. 자막 데이터 모델

```typescript
Subtitle {
  clipId: string
  segments: [{
    start: number        // 클립 내 상대 시간 (초)
    end: number
    text: string
    style?: SubtitleStyle  // 구독자 전용
  }]
}

SubtitleStyle {
  font: string
  color: string
  background: string
  position: 'top' | 'center' | 'bottom'
  animation: 'none' | 'fade' | 'typing' | 'bounce'
}
```

### Phase 3: 성장 엔진 (2~3주)

공유 → 유입 → 전환 루프를 만든다.

#### 3-1. 바이럴 장치

- [ ] 공유 페이지 하단 "나도 클립 만들기" CTA 버튼
- [ ] 공유 링크 조회수 통계 (만든 사람에게 동기 부여)
- [ ] 인기 공유 클립 디스커버리 페이지

#### 3-2. 커뮤니티 기능 (가벼운 수준)

- [ ] 공유 클립에 반응 (좋아요, 이모지 리액션)
- [ ] 채널별/게임별 태그 시스템
- [ ] 인기 클립 랭킹

#### 3-3. 크리에이터 도구

- [ ] 임베드 코드 생성 (블로그/커뮤니티 삽입용)
- [ ] 클립 모음 페이지 (여러 클립을 하나의 페이지로)
- [ ] 개인 클립 채널 프로필

---

### Phase 4: 확장 (장기)

- [ ] **GIF/숏폼 내보내기** — 클립을 GIF나 세로형 숏폼으로 변환
- [ ] **팀 워크스페이스** — MCN/방송국용 협업 공간
- [ ] **API 공개** — 외부 서비스 연동
- [ ] **모바일 앱** — PWA → 네이티브 앱
- [ ] **추가 플랫폼** — 아프리카TV, 네이버 LIVE 등
- [ ] **고급 AI** — 자동 챕터 생성, 감정 분석, 하이라이트 점수

## 홍보 계획

### 출시 전 (Phase 0~1)

- 랜딩 페이지 + 대기자 명단 오픈
- 티저 영상 제작 (ClipNote으로 하이라이트 추출하는 과정)
- 개발 블로그 연재

### 출시 직후 (Phase 1 완성 시점)

| 채널 | 전략 |
|------|------|
| 스트리머 협업 | 치지직/트위치 중소 스트리머에게 무료 제공, 방송 중 시연 |
| 커뮤니티 | 아카라이브, 디시인사이드, 레딧에 사용 사례 공유 |
| 숏폼 영상 | "3시간 방송 → 5분 하이라이트" 비포/애프터 |
| ProductHunt | 글로벌 런칭으로 해외 초기 사용자 확보 |

### 성장기 (Phase 2~3)

- SEO 콘텐츠: "스트리밍 하이라이트 만드는 법" 등 검색 유입
- 크리에이터 파트너십: 추천인 코드로 포인트 적립
- 교육 시장: 인프런, 클래스101 연동 제안
